3. Instrucciones de Debug

  1. Abrir herramientas de desarrollador (F12)
  2. Ir a la pestaña Console
  3. Hacer clic en cualquier enlace de tema
  4. Verificar en la consola:
    • Mensaje "Inicializando TemaPage..."
    • Mensaje "Cargando tema con ID: [id-del-tema]"
    • Mensaje "Respuesta del servidor: 200"
    • Mensaje "Tema cargado: [objeto-tema]"
    • Mensaje "Renderizando tema: [objeto-tema]"
    • Mensaje "FAQs renderizadas correctamente"
  5. Verificar visualmente:
    • Título del tema en rojo
    • Video de YouTube embebido
    • Sección de FAQs con preguntas clicables

4. Problemas Comunes

  • Si no se ve el video: Verificar que el iframe tenga src="https://www.youtube.com/embed/..."
  • Si no se ven las FAQs: Verificar que el contenedor #faqsContainer tenga contenido
  • Si hay errores en consola: Revisar la red (Network tab) para ver si la API responde
